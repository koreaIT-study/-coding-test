```java
public class Main{
    public static void main(String[] args){
      
    }
}

```
위처럼 적어야지 컴파일 에러가 나지 않음.

```java
public class Test{
    public static void main(String[] args){

    }
}
```
두번째처럼 class 이름을 Main이 아닌 다른 것으로 적으면 컴파일 에러가 일어남.
