# LVM
## LVM 개념 이해 
- LVM == Logical Volume Manager == 논리 하드디스크 관리자
- 여러 하드디스크를 필요에 따라 합치고 나눌 있음
<br>
<br>

주요 용어<br>
물리 볼륨(Physical Volume) : 실제 파티션을 말함
<br>

볼륨 그룹(Volume Group) : 물리 볼륨을 합쳐서 1개의 물리그룹으로 만든 것
<br>

논리 볼륨(Logical Volume) : 볼륨 그룹을 1개 이상으로 나눈 것(논리적 그룹)
<br>
